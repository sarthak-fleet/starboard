# Surfaces

Production URLs, API routes, and agent-indexing surfaces. The canonical list of
what the deployed Worker serves.

## Production

- **App:** <https://starboard.codevetter.com> (Cloudflare Worker `starboard`,
  custom domain).

## Primary routes

| Route | Role |
| --- | --- |
| `/` | Landing / redirect |
| `/stars` | Dashboard (filters, search, virtual grid) |
| `/explore/[...slug]` | Repo detail (comments, votes, likes, similar) |
| `/discover` | Seeded popular repos + tool facets |
| `/projects` | Fleet project list |
| `/projects/[slug]` | Fleet-aware recommendations for a project |
| `/lists/[slug]` | Public shared list (SSR) + `list.json` export |
| `/radar` | Maintainer/release signals |
| `/reports/[slug]` | Shareable insight report |
| `/stack-builder` | Stack composition helper |
| `/tools` | Tool intelligence aggregate |
| `/tools/[toolKey]` | Tool detail |
| `/about`, `/privacy`, `/terms` | Legal/marketing shell |

## Primary API

| Route | Role |
| --- | --- |
| `GET /api/stars` | Filtered stars + facets |
| `POST /api/stars/sync` | Sync starred repos from GitHub |
| `/api/repos/[repoId]/*` | Tags, list assignment, likes, comments, similar, star-history, save, tools |
| `/api/lists/*` | CRUD user lists + public list read |
| `/api/projects/*` | Fleet project list + recommendations |
| `GET /api/discover` | Seeded popular repos |
| `GET /api/radar` | Maintainer/release signals |
| `GET /api/growth` | Fastest-grower rankings |
| `GET /api/tools` | Tool intelligence aggregate |
| `/api/alerts/*` | Preferences, inbox, weekly |
| `GET /api/digest/weekly` | Weekly digest payload |
| `/api/reports/*` | Create / read shareable reports |
| `POST /api/embeddings/generate` | Embedding generation (admin) |

## Agent-indexing surfaces

Generated by `agent-edge.mjs` (do not hand-edit — regenerated by the fleet
agent-edge generator) and served from the Worker:

- `/llms.txt` — concise agent index.
- `/llms-full.txt` — full agent index.
- `/index.md` — markdown negotiation entry.
- `/api/ai` — structured agent JSON.
- `/robots.txt` — crawler directives.
- `/sitemap.xml` — site map.

These are the *app's* agent surfaces, distinct from the docs-site `llms.txt`
emitted by Blume (see [blume.config.ts](../../blume.config.ts)).

## Fleet snapshot

`data/fleet-projects.generated.json` is the checked-in fleet project snapshot
consumed by My Projects. Regenerate with `pnpm fleet:extract-projects` after
fleet `PROJECT_STATUS.md` or dependency changes.
